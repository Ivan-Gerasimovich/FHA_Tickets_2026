# FHA_Tickets — Билеты по ОВА

Этот проект представляет собой оцифрованную базу ответов на экзаменационные билеты по курсу **ОВА (Основы высшей алгебры)**. Материалы были перенесены из рукописных конспектов в структурированный цифровой формат.

## 🚀 О проекте

Цель проекта — создать удобный, читаемый и профессионально оформленный справочник для подготовки к экзамену. В общей сложности оцифровано 70 билетов, включая определения, теоремы и математические формулы.

## 🛠️ Технологии

Для верстки используется [Typst](https://typst.app/) — современная и быстрая альтернатива LaTeX. 

**Особенности оформления:**
- **Специальные блоки**: Для теорем, определений, лемм, следствий и замечаний созданы отдельные визуальные стили.
- **Автоматизация**: Нумерация билетов и формул обновляется автоматически.
- **Чистый дизайн**: Академический стиль с цветовым выделением логических блоков.

## 🌿 Ветки проекта (Версии оформления)

В репозитории доступны разные ветки, которые представляют собой различные варианты визуального оформления (стили в `conf.typ`):

- `main` — Основная версия оформления.
- `tiny` — Компактная/упрощенная версия стиля.
- `Herasimovich` — Альтернативная версия оформления (цветовая палитра)(аналогична `tiny`).

Чтобы переключиться на другую версию, используйте команду:
```bash
git checkout <название_ветки>
```

## 📁 Структура проекта (для ветки main)

- `FHA/Tickets/typ/` — **Основная ветка с кодом**. Здесь находится весь контент.
  - `chapters/` — Отдельные файлы для каждого билета (от `01.typ` до `70.typ`).
  - `conf.typ` — Конфигурационный файл с настройками стилей и макетов.
  - `main.typ` — Главный файл, который собирает все билеты в один документ.

## 📖 Как скомпилировать в PDF

Чтобы превратить исходный код в красивый PDF-документ:

1. **Установите Typst**: Инструкции доступны на официальном сайте [typst.app](https://typst.app/).
2. **Запустите компиляцию**: Откройте терминал в корне проекта и выполните команду:
   ```bash
   typst compile FHA/Tickets/typ/main.typ
   ```
3. **Результат**: В папке с исходниками появится файл `main.pdf`.
